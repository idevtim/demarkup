## Highlights

- Initial public release of DeMarkup
- Convert any webpage to clean, well-structured Markdown
- Three conversion modes: main content extraction, full page, and selection
- GitHub Flavored Markdown support (tables, strikethrough, task lists)
- Smart content cleaning removes navigation, ads, and noise
- Front matter with page metadata (title, URL, date)
- Token count estimation for LLM context planning
- Keyboard shortcut (Cmd+Shift+M) and context menu support
- Configurable options for headings, links, code blocks, and more
- Debug logging with export capability
